<p align="center">
    <b>This application processes structured .txt files containing cooking recipes and allows users to search them.</b>
</p>

<h2 id="features">🚀 Features</h2>

- list - lists the recipes
- stop - stops the program
- find name - searches recipes by name
- find cooking time - searches recipes by cooking time
- find ingredient - searches recipes by ingredient

<h2 id="structure">🏗️ Project Structure</h2>

```bash
src/main/java/com/pedrohlopes
├── Main.java        # Main Class
├── Recipe.java      # Store recipe atributes
├── RecipeFile.java  # Read recipe file and add recipes
├── UserInterface    # User interface structure
```
